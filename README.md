# Inventory Master

Inventory Master is a Minecraft mod designed to simplify inventory and storage management.

The goal is simple: spend less time organizing chests and more time playing the game.

## 🎮 Minecraft Version

- **Minecraft:** 1.21.1
- **Loader:** Fabric
- **Java:** 21+

## ✨ Features

### 📦 Automatic Sorting
- Merges item stacks
- Groups similar items together
- Intelligently arranges inventory slots

### ⭐ Favorite Slots
- Lock specific inventory slots
- Favorite items always stay in place during sorting

### 🗂️ Item Categorization
- Automatically sorts items into 7 categories:
  - Building Blocks
  - Ores & Resources
  - Valuable Items
  - Food
  - Tools
  - Armor
  - Miscellaneous

### 🎒 Virtual Bags
- Browse items by category with dedicated tabs
- Cleaner inventory experience
- Quick access to specific item types

### 🏠 Smart Deposit System
- Remember designated storage chests
- Assign chests to specific item categories
- One-click deposit of all items to correct chests

### 🔍 Instant Search
- Built-in search bar for quick item lookup
- Fuzzy matching support
- Works across all categories

## 🚀 Development

### Setup

1. Clone the repository
2. Run `./gradlew build` (Windows) or `./gradlew build` (Linux/Mac)
3. Run `./gradlew genSources` to generate source mappings
4. Open in IDE and run with Fabric development workspace

### Building

```bash
./gradlew build
```

The mod JAR will be in `build/libs/`

### Running

Use the Fabric development workspace to test the mod in-game.
